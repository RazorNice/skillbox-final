# Skillbox infrastructure

Этот репозиторий содержит скрипты и конфигурации для настройки собственного VPN-сервера с OpenVPN и Easy-RSA.

## Структура(буду обновлять постепенно)

- `openvpn-ca-config.deb` - пакет для установки конфигов OpenVPN
- `ca-config.deb` - пакет для установки и настройки easy-rsa
- `startsetup.sh` — скрипт начальной настройки сервера
- `README.md` — описание проекта (этот файл)
